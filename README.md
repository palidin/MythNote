## 📖 关于 MythNote

**MythNote** 是一款专为注重隐私与长期可控性的用户设计的本地笔记应用。

我们相信：

> 笔记应该像纸质日记一样，只属于你自己。

因此，MythNote 不依赖任何商业云服务，所有笔记**仅存储在本地**，并由应用自动完成版本管理与跨设备同步。你始终拥有 100% 的数据主权。

---

## 🍃 为什么选择 MythNote

在充斥着广告、订阅和数据扫描的云笔记时代，MythNote 提供了一种更克制、更透明的选择。

| 维度   | **MythNote**        | 印象笔记  | Obsidian  |
| ---- | ------------------- | ------ | -------------- |
| 数据主权 | **完全掌控**，本地存储 + Git | 厂商控制   | 本地存储           |
| 隐私安全 | **极高**，无云端扫描        | 存在隐私风险 | 高              |
| 功能体验 | **轻量纯粹**，专注写作       | 功能臃肿   | 插件依赖重          |
| 版本记录 | **自动版本管理**，完整可追溯    | 通常需付费  | 需额外配置          |
| 同步方式 | **免费透明**，私有仓库       | 闭源同步   | 复杂或付费          |

## 🎯 MythNote 适合谁？

✅ 你可能会喜欢 MythNote，如果你：

- 重视**数据隐私与完全控制权**
- 希望笔记**纯本地存储，不依赖商业云**

## ✨ 功能特性

### 📝 专业的 Markdown 体验

* **完整支持 GFM**（GitHub Flavored Markdown）
* **实时预览**，专注写作不中断思路
* **代码高亮**，支持多种主流编程语言

### 🏷️ 灵活的标签系统

* **无限层级标签**：`工作/项目A/进度`
* **多标签归属**：一篇笔记可属于多个分类
* **全局搜索**：基于标签与元数据的快速定位

### 🔄 自动化版本管理

* **自动版本存档**：每次保存即生成历史记录
* **完整历史追踪**：可回溯任意修改记录
* **跨平台一致性**：Windows / macOS / Linux 统一体验

## 🚀 快速开始

### 系统要求

* **.NET 8.0** 或更高版本
* **Git**（用于版本管理与同步）

### 安装与初始化

1. **下载应用**
   前往 [Releases](https://github.com/palidin/MythNote/releases) 下载对应系统的安装包。

2. **首次登录**
   启动后浏览器将自动打开本地管理界面。

   初始默认账号（仅用于首次初始化）：

   * 用户名：`mythnote`
   * 密码：`mythnote`

   ⚠️ **请在首次登录后立即修改密码**。

3. **配置备份仓库**
   在设置页面填写你的 **仓库地址** 与 **访问 Token**。

   配置完成后，所有笔记将自动获得安全的版本备份与跨设备同步能力。

## 📚 笔记存储格式

MythNote 使用 **标准 Markdown + Frontmatter**，确保数据高度可迁移、可长期保存：

```markdown
---
title: 我的深度思考
tags: [学习, 哲学]
created: '2026-02-07T18:00:00'
modified: '2026-02-07T18:30:00'
---

# 标题
正文内容...
```

你可以随时使用其他 Markdown 工具直接读取或迁移这些文件。
