## 📖 关于 MythNote

MythNote 是一款专为注重隐私的用户设计的笔记管理工具。我们认为，笔记应该像日记本一样私密，而不应成为商业服务的附属品。在这里，你的每一行文字都完全存储在本地，并由你亲自掌控的 Git 仓库提供备份。

---

## 🍃 为什么选择 MythNote？

在喧嚣的云笔记市场中，我们为你保留了一份宁静与掌控感。

| 维度 | **MythNote (我们的坚持)** | **传统商业云笔记** | **Obsidian 等本地工具** |

## ✨ 功能特性

### 📝 专业的 Markdown 体验

* **完整支持 GFM** - 兼容 GitHub Flavored Markdown 标准。
* **实时预览** - 所见即所得，让你的思绪不再被标记符中断。
* **智能高亮** - 支持多种编程语言的代码块高亮。

### 🏷️ 灵动的标签管理系统

* **无限层级** - 支持 `工作/项目A/进度` 这种树状层级，结构再复杂也能井井有条。
* **多重分类** - 一篇笔记可属于多个标签，摆脱传统目录的局限性。
* **全局搜索** - 基于标签与元数据的闪电搜索，瞬间定位灵感。

### 🔄 自动化的版本管理

* **Git 自动化** - 每次保存都会自动生成版本存盘，无需手动提交。
* **历史追踪** - 完整的修改记录，支持查看过去任何一个时间点的笔记内容。
* **跨平台一致** - 一套代码支撑 Windows、macOS 和 Linux，体验高度统一。

## 🚀 快速开始

### 系统要求

* **.NET 8.0** 或更高版本。
* **Git** 环境（用于笔记版本控制）。

### 安装与初始化（仅需三步）

1. **获取应用**：前往 [Releases](https://github.com/your-username/MythNote/releases) 下载适合你系统的安装包。
2. **安全登录**：启动后浏览器会自动打开，请使用默认凭据登录：
* **用户名**: `mythnote`
* **密码**: `mythnote`


3. **配置备份**：在设置页面填入你的 **Git 仓库地址** 和 **访问 Token**。
* *配置完成后，你的笔记将获得 Git 带来的自动化备份与同步能力。*

## 📚 使用指南

### 笔记存储格式

MythNote 采用标准的 Markdown + Frontmatter 格式，确保你的数据具有极高的可迁移性：

```markdown
---
title: 我的深度思考
tags: [学习, 哲学]
created: '2026-02-07T18:00:00'
modified: '2026-02-07T18:30:00'
---

# 标题
正文内容...

```
